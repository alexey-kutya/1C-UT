Настройки системы

Форма "Настройки системы" предназначена для установки основных параметров, определяющих правила работы системы, доступность того или иного функционала.

На что влияет значение поля "Заголовок программы"?

Заголовок программы позволяет задать произвольный заголовок, который будет отображаться в заголовке основного окна программы.

При этом в заголовке также указывается текущий пользователь программы и информация о текущем проекте, если он был выбран.

Для чего используется "Основной календарь"?

Основной календарь используется для расчета длительности и дат контрольных точек в технических проектах

Для чего можно использовать в уведомлениях внешние ссылки?

Уведомления системы (о направленных пользователям ошибках, выполнении технических проектов и т.д.) рассылаются в виде почтовых сообщений по электронной почте. При этом в тексте сообщения для удобства работы содержатся ссылки на объекты СППР, которых касаются уведомления (технические проекты, ошибки...).

При условии публикации базы, из которой выполняется рассылка, на веб-сервере, по ссылке можно непосредственно перейти к нужной информации, не прибегая к дополнительным действиям. Для этого ссылка должна включать расположение информационной базы.

При установленном флаге "Использовать в уведомлениях внешние ссылки" доступно поле "Адрес публикации информационной базы на веб-=сервере". Адрес, указанный в данном поле, используется системой для формирования ссылки в тексте уведомления.

Как используется поле "Признаки получения информации об ошибках от внешних источников"?

При регистрации ошибок может указываться исходная информация об ошибках (текст сообщения об ошибке, текст почтового сообщения пользователя с описанием ошибки и т.д.).
При необходимости ошибки, информация о которых получена из внешних источников, можно помечать специальном флагом, чтобы отделять от ошибок, полученных от внутренних источников.

При этом признак получения ошибки от внешних источников может быть установлен автоматически на основании данных, содержащихся в исходном описании ошибки. Если в исходном описании содержится текст, указанный в настройках для идентификации внешних источников, признак устанавливается автоматически.
