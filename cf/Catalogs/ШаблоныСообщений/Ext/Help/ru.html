Справочник «Шаблоны сообщений»

 

Справочник содержит перечень шаблонов, которые используются при отправке сообщения ответственному лицу по электронной почте или sms.

 

Реквизиты формы элемента

Форма элемента справочника включает следующие реквизиты:

  • Наименование — название шаблона, основное представление элемента справочника;
  • Код — уникальный идентификатор элемента справочника, автоматически формируемый системой.

Закладки «Шаблон электронного письма» и «Шаблон SMS»

  • Заголовок — заголовок электронного письма или короткого сообщения;

  • Текст сообщения — текст электронного письма или короткого сообщения с возможностью использования тегов.

На закладке «Шаблон электронного письма» текст сообщения можно форматировать с помощью кнопок командной панели в нижней части формы. Эта опция недоступна в веб-клиенте.

 

Предопределенные элементы

Список используемых шаблонов:

  • Изменение ответственного по запросу на изменение — шаблон отправляемый сотруднику установленного в качестве ответственного для документа «Запрос на изменение»;
  • Изменение ответственного по обращению — шаблон отправляемый сотруднику установленного в качестве ответственного для документа «Обращение»;
  • Изменение ответственного по наряду — шаблон отправляемый сотруднику установленного в качестве ответственного для документа «Наряд»;
  • Изменение ответственного по проблеме — шаблон отправляемый сотруднику установленного в качестве ответственного для документа «Запрос на изменение»;
  • Изменение рабочей группы по запросу на изменение — шаблон отправляемый сотруднику / сотрудникам рабочей группы установленной в качестве ответственной для документа «Запрос на изменение»;
  • Изменение рабочей группы по инциденту — шаблон отправляемый сотруднику / сотрудникам рабочей группы установленной в качестве ответственной для документа «Обращение»;
  • Изменение рабочей группы по проблеме — шаблон отправляемый сотруднику / сотрудникам рабочей группы установленной в качестве ответственной для документа «Проблема»;
  • Оповещение по задаче — оповещение сотрудника или контактного лица по сформированной задаче;
  • Оповещение по приоритету — формирование сообщения группе сотрудников назначенной для приоритета.
