Функции

Функция – совокупность действий (операций), выполняемых пользователями и внутренними механизмами проектируемой информационной системы (ИС) и связанных с преобразованием информации. Список функций ИС определяется в результате декомпозиции – разбиения функции на более простые составляющие (функции более низкого уровня).

Результатом проектирования функции являются:

  • схема функции,
  • описание функции,
  • описание объектов схемы,
  • декомпозиция функции на дочерние функции.

Иерархия функции опеределяется в дереве функций, после чего функции переносятся на IDEF0-схемы для уточнения информационных потоков (объектов данных).
Функции увязывают информацию о возможностях, поддерживаемых системой для реализации автоматизируемых процессов, потоках данных, обрабатываемых в информационной системе и исполнителях, исполняющих процессы в рамках конкретных функций.

Функции в дереве согласуются между собой при помощи связей. Согласованность функций определяет корректность декомпозиции функций. Система поддерживает проверку корректности функций на основании формальных правил проектирования.
