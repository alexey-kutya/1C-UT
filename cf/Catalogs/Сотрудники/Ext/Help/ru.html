Справочник «Сотрудники»

Выбор: Управление персоналом → Сотрудники

 

Справочник «Сотрудники» предназначен для хранения данных о сотрудниках предприятия. Справочник связан со справочником «Физические лица» в отношении «один-к-одному».

 

Реквизиты формы элемента

  • ФИО — фамилия, имя и отчество сотрудника, основное представление справочника;
  • Код — уникальный идентификатор элемента справочника, автоматически формируемый системой;
  • Комментарий — произвольная словесная характеристика сотрудника.

 Закладка «Общая информация»

  • Персональные данные — группа реквизитов, характеризующая личные данные сотрудника:
    • Физ. лицо — элемент справочника «Физические лица», данные которого характеризуют сотрудника как физическое лицо.
    • Пол — пол сотрудника;
    • Дата рождения — дата рождения сотрудника;
    • Фото — поле для добавления фотографии сотрудника.
  • Кадровые данные — группа реквизитов, характеризующая кадровые данные сотрудника:
    • Организация — организация, в которой трудоустроен работник;
    • Подразделение — подразделение организации, в котором состоит сотрудник;
    • Должность — должность, занимаемая сотрудником;
    • Рабочее место — рабочее место, закрепленное за сотрудником;
    • Уволен — признак того, что сотрудник был уволен из организации;
    • График — график работ, назначенный сотруднику.

 

 Закладка «Контактная информация»

Закладка «Контактная информация» содержит перечень способов связи с данным сотрудником. Виды связи выбираются из справочника «Виды связи». Для видов контактной информации, отмеченных в списке как основные, в системе выполняется отправка уведомлений. Для отметки вида связи в качестве основного предназначена кнопка «Установить основным».

 Закладка «Компетенции»

Закладка «Компетенции» доступна, только если в параметрах системы установлена отметка «Использовать учет компетенций». На закладке можно назначить компетенции для сотрудника из справочника «Компетенции». Перечень услуг, которые может оказывать сотрудник, определяется автоматически и предваряется зеленым сигналом в списке услуг. Услуги, которые сотрудник оказывать не может, отображаются с красным или желтым сигналом. Для таких услуг и их компонент в нижнем поле группы «Услуги» выводится список недостающих компетенций.

 Закладка «Стоимость ресурса / Мотивация»

Закладка «Стоимость ресурса / Мотивация» доступна только при установленной отметке «Управление финансами» в параметрах системы. На закладке можно заполнить реквизиты:

  • Стоимость — стоимость часа работы сотрудника в рублях за час;
  • Статья расходов — наименование статьи расходов для учета трудозатрат сотрудника;
  • База расчета мотивации — значение, которое используется при расчете итогового коэффициента по KPI. Может использоваться как коэффициент или как базовая сумма премиальных.
