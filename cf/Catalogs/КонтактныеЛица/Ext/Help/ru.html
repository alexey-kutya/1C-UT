Справочник «Пользователи (потребители услуг)»

Выбор: Управление каталогом и уровнем услуг → См. также → Пользователи (потребители услуг)

 

Справочник «Пользователи (потребители услуг)» подчинен справочнику «Клиенты» и предназначен для хранения перечня контактных лиц каждого из клиентов.

 

Реквизиты формы элемента

Справочник реализован на форме, содержащей следующие закладки: «Общая информация», «Контактная информация», «Доступ к КЕ».

На закладке «Общая информация» находятся общие реквизиты контактного лица, такие как:

  • Физическое лицо — ссылка на справочник Физических лиц;
  • Дата рождения — недоступен для редактирования, автоматически отображается дата рождения Физ. лица;
  • Пол — недоступен для редактирования, автоматически отображается пол Физ. лица;
  •  Клиент — клиент, чьё контактное лицо описывается;
  • Категория — категория пользователя;
  • Должность — должность контактного лица;
  •  Рабочее место — рабочее место контактного лица;
  • CN — реквизит, обеспечивающий уникальность контактного лица для синхронизации Active Directory;
  • Комментарий — произвольная словесная характеристика пользователя.

На закладке «Контактная информация» перечисляются способы связи с данным контактным лицом. Может быть указан электронный адрес или указать номер мобильного телефона, на которые будут отправляться извещения о состоянии инцидентов.

  Примечание: поля с типом связи «E-Mail» используются при отправке уведомлений; поля с типом связи «Телефон» используются для отправки SMS. Следует иметь в виду, что для корректной работы обработки загрузки данных необходимо задать основную контактную информацию с типом «E-Mail».

На закладке «Доступ к КЕ» хранятся параметры доступа пользователя к КЕ в разрезе различных параметров.
